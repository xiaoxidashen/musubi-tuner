# WAN 2.2 训练采样提示词
# 格式：提示词 --w 宽 --h 高 --f 帧数 --s 步数 --l CFG_scale --d 种子
#
# 参数说明：
# --w: 宽度（默认 256）
# --h: 高度（默认 256）
# --f: 帧数（默认 1）
# --s: 推理步数（默认 20）
# --l: CFG scale（Wan 默认 5.0）
# --d: 随机种子（默认随机）
# --n: 负面提示词（可选）
# --fs: discrete flow shift（可选，默认 14.5）

# 示例 1: 基础测试
lh戴着星星形状的眼镜看着镜头 --w 512 --h 512 --f 1 --s 25 --l 5.0 --d 42
